Интерполяция
{{ variable }}

Вывод различных типов переменной.

String: {{ name }}
Sum: {{ 90 + 30 }}
Numeber: {{ numeber }}
Method: {{ functionName() }}
If statement: {{ true ? 'I am OK' : '' }}
Динамические атрибуты

v N1

v-bind:href="variable"

v N2

:href="variable"
:src="'/path/path/' + varoable"
:class="{ className: isTrue }"
:style="{ fontSize: size + 'px' }"
Вывод HTML кода
new Vue({
  data: {
    variableName: '<a href="https://www.google.com/">Google</a>'
  }
})
 
<div v-html="variableName"></div>
Добавление событий

v N1

<div v-on:click="functionName"></div>

v N2

<div @click="functionName"></div>
 
new Vue({
  methods: {
    functionName: function(){
      //code
    }
  }
})
Передача параметров в метод
<div @click="functionName(10, $event)"></div>
 
new Vue({
  methods: {
    functionName: function(n, event){
      //code
    }
  }
})
Модификаторы событий
<div @click.prevent="functionName(10, $event)"></div>
Модель для работы с формами
<input v-model="variableName">
Работа с CSS классами
<div :class="variableName"></div>
<div :class="{'className': isTrue, 'className': !isFalse}"></div>
<div :class="[classNameA, classNameB]"></div>
<div :class="[classNameA, { classNameB: isTrue, classNameC: isTrue }]"></div>
Динамические стили
<div :style="{'background': color, 'height': height + 'px', 'width': height + 'px'}"></div>
Управление отображением элементов

v N1

<div v-if="false === true"></div>
<div v-else-if="false === true"></div>
<div v-else></div>

v N2

<div v-show="isTrue"></div>
Работа со списками
<div v-for="(value, key, index) in arrayName"> {{index + 1}} {{key}} {{value}} </div>
Оптимизация приложения с computed
new Vue({
  computed: {
    functionName: function(){
      return 1 + 3
    }
  }
})
Отслеживание изменений с watch
new Vue({
  data: {
    variableName: 0
  },
  watch: {
    variableName: function(val, val2){
      //code
    }
  }
})
Доступ к DOM элементам (ref)
<div ref="refName"></div>
 
new Vue({
  data: {
    variableName: 0
  },
  methods: {
    functionName: function(){
      this.$refs.refName
    }
  }
})
